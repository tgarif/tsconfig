# @tgarif/tsconfig

[![npm](https://img.shields.io/npm/v/@tgarif/tsconfig?color=a1b858&label=)](https://npmjs.com/package/@tgarif/tsconfig)

## Usage

### Install

```bash
npm i -D @tgarif/tsconfig
```

### Config `tsconfig`

```json
{
  "extends": "@tgarif/tsconfig/tsconfig.json",
  "rules": {}
}
```
